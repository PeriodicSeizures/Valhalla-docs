# RouteManager

### `RouteManager:InvokeView(target, zdoid, repr, ...)`
  > Invokes a RoutedRpc aimed towards a peer(s)

### `RouteManager:Invoke(target, repr, ...)`
  > Invokes a RoutedRpc aimed towards a peer(s)

### `RouteManager:InvokeAll(repr, ...)`
  > Invokes a RoutedRpc aimed towards all peers
