# Mod

### `mod.name`
  > Returns `string`
  
  > The name of the mod
  
### `mod.version`
  > Returns `string`
  
  > The version of the mod
  
### `mod.apiVersion`
  > Returns `string`
  
  > The api version of the mod
  
### `mod.description`
  > Returns `string`
  
  > The description of the mod
  
### `mod.authors`
  > Returns `container<string>`
  
  > The authors of the mod
