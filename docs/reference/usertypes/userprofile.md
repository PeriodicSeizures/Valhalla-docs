# UserProfile

A structure to hold user data. 

### `UserProfile.new(name, ign, nid)`
  > Returns `UserProfile`
  
  > Constructs a `UserProfile` from a username, in-game name, 
  and network-id.
  
### `userProfile.name`
  > Returns `string`
  
  > The name of the profile
  
### `userProfile.ign`
  > Returns `string`
  
  > The in-game name of the profile
  
### `userProfile.nid`
  > Returns `string`
  
  > The network-id of the profile
