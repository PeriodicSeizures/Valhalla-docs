# ZDOID

A ZDO identity structure

### `ZDOID.new(uuid, id)`
  > Returns `ZDOID`

  > Constructs a ZDOID from a `int64` uuid and a numeric id
  
### `ZDOID.NONE`
  > A zero ZDOID (0, 0)
  
### `zdoid.uuid`
  > Returns `Int64`
  
  > The uuid of the zdoid as a `int64`
  
### `zdoid.id`
  > Returns `number`
  
  > The id of the zdoid
